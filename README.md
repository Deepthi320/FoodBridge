# 🍲 FoodBridge

FoodBridge is a web platform that connects food donors with people in need in real-time, helping reduce food waste and support communities.

## 🚀 Features
- Donor can add food details (name, quantity, location, contact)
- Users can view available food instantly
- Mark food as taken (real-time update)
- 🚨 Emergency option for urgent food (near expiry)
- Reduces food waste and supports needy people

## 🌍 SDG Contribution
Supports Zero Hunger and Responsible Consumption by minimizing food wastage and connecting communities efficiently.

## 🛠️ Tech Used
- HTML
- CSS
- JavaScript
- Firebase

## 📌 Future Improvements
- Location-based filtering
- Automatic expiry timer
- Data dashboard (total food donated, meals served)
